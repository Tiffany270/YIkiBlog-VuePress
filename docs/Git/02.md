# Merge and Rebase
![merge](./imgs/01.png)
- 使用` git merge `来做分支合并。
- 将`master`分支合并到`feature`分支上，会新增一个`commit`节点来记录这次合并
- `Git`会有很多合并策略，其中常见的是`Fast-forward`、`Recursive `、`Ours`、`Theirs`、`Octopus`
- 默认`Git`会帮你自动挑选合适的合并策略，如果你需要强制指定，使用`git merge -s <策略名字>`
